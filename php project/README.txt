
How to Connect to MySQL Server:
1. In the db_connect.php file, update the following settings with your credentials:
   - username: Your MySQL username
   - servername: The MySQL server address (e.g., localhost or your remote server's IP)
   - password: Your MySQL password
   - dbname: The name of your database

How to Access the Admin Control Panel:
1. After creating a user, set the is_admin column to true for that user in the database.
2. If you are already logged in, log out and then log back in to access the admin panel.
